## Libraries Used


    1.expo             -> Base framework for the entire app
    
    2.react-native     -> Core UI components
    
    3.react-native-animatable	   -> For animations (fade, slide, pop-up)
    
    4.react-native-safe-area-context	-> Handles safe area padding
    
    5.expo-font     -> Loads custom Google Fonts
    
    6.expo-status-bar    -> Controls the device status bar
    
    7.@expo/vector-icons    -> For Icons
